 AI Skin Care Assistant

 AI skin care assistant is a smart web -based application that helps users analyze their skin condition and get skincare suggestions using Artificial Intelligence
 and Machine learning teachniques.The system detects different skin-related issues and provides personalized skincare recommendations to users.


FEATURES:

skin type detection.
AI based skin analysis.
Personalized skincare suggestions.
Users-friendly interface.
Smart recommendation system


Technologies Used :

python
Machine Learning
html
css
opencv
streamlit
sklearn


How it works:

Users uploads an image or enters skin details.
The AI model analyzes the skin condition.
The system predicts possible skin type/issues
Personalized skin care recommendations are displayed 

Project Objective:

The main objective of this project is to help users understand their skin condition and get basic skincare guidance using AI technology
